# 最优化理论

老师的讲课视频在 B 站[BV1nB4y137j9](https://www.bilibili.com/video/BV1nB4y137j9)。
主要分为数学基础、局部最优化问题求解和全局局部最优化问题，只讲了局部的部分。

- 绪论
- 无约束单变量优化问题——线搜索
  - 单峰区间的确定
  - 黄金分割法：线性收敛
  - 牛顿法：超线性收敛
  - 抛物线法
  - 非精确线搜索方法 Wolfe 准则，Armijo 准则
- 无约束多维优化问题
  - 梯度法（最速下降法）与牛顿法
  - 共轭方向法：G 共轭、二次终止性、共轭梯度法
  - 拟牛顿法：BFGS、DFP、Broyden 族
- 约束优化问题：约束问题的最优性条件（一阶必要，二阶充分）
  - 可行方向法：Zoutendijk（转化为**线性规划**问题）、 梯度投影法（边界上时，向边界投影），~~简约梯度法~~
  - 罚函数法：外罚函数法、内点法、乘子法
